**THiNK English 分级测试成绩报告（家长版）**
*(Cambridge A1–A2 Placement Test)*

---

### 1. 测试简介

本测试总分 **100 分**，分为四个部分：

| 部分                              | 题量      | 分值   | 主要考查            |
| ------------------------------- | ------- | ---- | --------------- |
| Grammar & Vocabulary            | 30 题    | 30 分 | A1–A2 关键语法、核心词汇 |
| Reading Comprehension           | 5 题     | 20 分 | 熟悉题材短文的主旨与细节理解  |
| Functional Language             | 选择+情景写作 | 20 分 | 日常情境用语与对话连贯性    |
| Basic Writing Skills            | 2 题     | 30 分 | 简短话题写作（兴趣、理由等）  |
| 以上比例保证听说读写综合考查，侧重基础语法词汇和真实场景表达  |         |      |                 |

| 成绩区间（100 分制）   | 欧标参考           | 推荐路径                                                                             | 具体说明                                                                                       |
| -------------- | -------------- | -------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------ |
| **0 – 39 分**   | Pre-A1 ～ A1 入门 | **先夯实基础**<br>• 语音/自然拼读<br>• 600+ 高频词<br>• be 动词、一般现在时等零起点语法                      | 成绩未达 40 分，正确率不足 40%，说明 A1 必备知识尚不牢固。建议先用启蒙资源（Kid’s Box、自然拼读课程等）打好发音和词汇基础，再衔接 Think Starter  |

### 4. 温馨提示给家长

1. **保持输入**：每天 15 分钟英文听读（校内外读物 + Think 音频）是提升词汇与语感的关键。
2. **适量输出**：写作与口语任务应紧扣学生兴趣，鼓励用 *because / and / but* 等连接词扩句。
3. **阶段评估**：每完成 4 个单元建议做一次小测，及时查看孩子的语法与词汇薄弱项并调整节奏。
4. **正向反馈**：家长可用“听到你用了新句型，我很惊喜！”等积极语言，帮助孩子建立英语自信。
